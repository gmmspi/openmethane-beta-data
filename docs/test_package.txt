the tests package has 2 main scripts.

test_cost_verbose:
- replicates the cost function and saves the output of every step to <archive_dir>/tmp_cost_verbose/

test_grad_verbose:
- replicates the gradient function and saves the output of every step to <archive_dir>/tmp_grad_verbose/

select which test to run by modifing the submit.sh script and submit it as a job
> qsub submit.sh
