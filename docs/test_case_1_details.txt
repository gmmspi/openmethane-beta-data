Math:
  1 kg Carbon / (year*m^2)
  = 1000 {kg/g} * (1/12.01) {mol/g} * {mole Carbon} /
      ( m^2 * 365.25 {days/year} * 3600 {s/day} )
  = 2.6384E-6 mol/(s*m^2)
  *note* mol(carbon) == mol(CO2)
therefore:
  EMIS_MAX = 2.6384E-4 mol/(s*m^2)
  emission sd = 1.3192E-5  mol/(s*m^2)
